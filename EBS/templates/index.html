Amazon Elastic Block Store (EBS) provides block-level storage volumes for use with EC2 instances.

  • Volume Types: gp2 (General Purpose SSD), gp3 (General Purpose SSD), io1 (Provisioned IOPS SSD), io2 (Provisioned IOPS SSD), st1 (Throughput Optimized HDD), sc1 (Cold HDD)
  • States: Creating, Available, In-use, Deleting, Deleted, Error
  • Note: Volumes can only be deleted when in 'available' state
